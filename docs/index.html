What does this program do?

04 — 예측 모델 / Prediction Model

앙상블 예측 구성

Ensemble prediction architecture

단기 — 1개월 후

Short-term · 1 Month · XGBoost

RSI · MA 비율 · 5/20일 수익률 · 변동성을 특성으로 사용. 분위수 회귀(10th·50th·90th)로 80% 예측 구간 산출. 지표 조합 패턴을 비선형 학습해 선형회귀보다 높은 정확도.

Uses RSI, MA ratios, momentum and volatility as features. Quantile regression (10th/50th/90th) for 80% prediction interval. Non-linear pattern learning.

중·장기 — 3 / 6 / 12개월 후

Mid & Long-term · Prophet (Meta)

트렌드 + 주간 계절성 + 연간 계절성 분해 학습. changepoint_prior_scale=0.05로 안정형 포트폴리오에 적합한 보수적 변화점. 12개월에는 배당률 반영 총수익률 추가 산출.

Decomposes trend + weekly + yearly seasonality. Conservative changepoint setting (0.05) suited for stable dividend assets. 12M adds dividend-inclusive Total Return estimate.

구간 / Period 모델 / Model 예측 구간 / Interval 배당 투자 활용 / Investment Use
1개월 후 / 1MXGBoost80% 분위수 회귀이번 달 DCA 매수 타이밍 참고
3개월 후 / 3MProphet80% 신뢰 구간다음 분기 배당 전후 포지션 점검
6개월 후 / 6MProphet80% 신뢰 구간반기 리밸런싱 기준 참고
12개월 후 / 12MProphet + 배당80% 신뢰 구간연간 자산 성장 + 총수익률 목표 점검

단기(1개월) 예측은 시장 노이즈로 인해 신뢰도가 낮습니다. 배당 성장 전략 특성상 6개월·12개월 예측이 실질적으로 더 유용합니다.

Short-term (1M) predictions carry higher uncertainty due to market noise. For dividend growth strategies, the 6M and 12M forecasts are more actionable.

06 — 자동화 / Automation

Windows 작업 스케줄러

Windows Task Scheduler — daily at 07:00 KST

미국 장 마감(한국 오전 5:00) 이후 2시간 버퍼로 당일 종가가 완전히 반영된 후 분석합니다. PC가 꺼져 있어도 켜지는 즉시 실행됩니다.

2-hour buffer after US market close (05:00 KST) ensures same-day closing prices are fully available. Runs on wake if PC was off at scheduled time.

# 관리자 PowerShell에서 실행 / Run in Administrator PowerShell
$action  = New-ScheduledTaskAction `
    -Execute  "C:\...\Python311\python.exe" `
    -Argument "main.py" `
    -WorkingDirectory "C:\...\Trading_Bot"

$trigger  = New-ScheduledTaskTrigger -Daily -At 7:00AM
$settings = New-ScheduledTaskSettingsSet -StartWhenAvailable -RunOnlyIfNetworkAvailable -ExecutionTimeLimit (New-TimeSpan -Hours 2)

Register-ScheduledTask -TaskName "IVV_Trading_Bot" `
    -Action $action -Trigger $trigger -Settings $settings -RunLevel Highest -Force

결과물 / Outputs

  • 노션: 날짜별 리포트 페이지 자동 생성Notion: dated report page auto-created daily
  • 이메일: 과매수/과매도 신호 + 노션 링크Email: RSI signal summary + Notion report link
  • 콘솔: 종목별 요약 테이블 출력Console: per-ticker summary table
  • 로그: logs/run_log.txt 실행 기록 누적Logs: cumulative run history in logs/run_log.txt

실행 흐름 / Run Order
